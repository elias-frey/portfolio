Data Analysis: Bicycle Webshop

Project Overview

This project serves as an example of how modern tools like Power BI can assist in visualizing data and generating valuable insights about the business.

The database containing information about bicycle webshop transactions, clients, and other descriptive data is stored in SQL Server. Using a series of SQL scripts, data was validated, cleaned, and transformed, after which it was exported to Power BI for visualization purposes.

GOAL

Create an interactive interface for the sales department and strategic decision-makers that provides an overview of the webshop's performance and includes insights into the client base and the most demanded products.

METHOD

The data, stored on SQL server is managed through a series of SQL scripts before being exported to PowerBI, where further analysis and design of a dashboard is done.

RESULT

An interactive PowerBI dashboard that consolidates data into a 3-page report. The main page is dedicated to an overview of business performance, the second to customer insights, and the third to details about the product range.
