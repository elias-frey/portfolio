Data Analysis: Bike Sharing in London

Project Overview

This project serves as an example of how data analysis can transform raw data into valuable insights about the business, uncovering trends and patterns that assist decision-makers and consequently enhance the overall efficiency of the venture.

The source data for this project comprises historical records of bicycle rental transactions that occurred in London over a two-year period. This dataset carries detailed information about the time the rental deal occured and under which weather conditions.

GOAL

The goal is to present the collected data in a way that effectively communicates valuable insights, which can be leveraged to improve the operational efficiency of the business.

METHOD

In this data analysis project, I employ Python for data preparation and utilize PowerBI for data visualization.

RESULT

The resulting dashboard unveils insights that can be taken into consideration when determining the fleet size for bicycle rentals, scheduling maintenance, and managing other assets, including the workforce.

Dashboard

After completing all preparations, the data was exported to PowerBI for in-depth analysis and visualization. The resulting interactive dashboard highlights:

  • Seasonal patterns in bike-sharing
  • Peak rental hours during the day
  • Optimal weather conditions for bicycle rentals
  • Rental activity distribution throughout the week
